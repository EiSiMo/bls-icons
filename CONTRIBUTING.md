# Contributing

I self-host this repository on my own [Forgejo instance](https://git.moritz.run)
because I want to stay in control of my data instead of relying on GitHub. The
downside: I don't open public account registration, which makes it hard for
others to contribute.

If you'd still like to help: issues, comments, or pull requests - write me an
email and I will create an account for you:

**[contribute@moritz.run](mailto:contribute@moritz.run)**

AI-generated code and issues are welcome.
